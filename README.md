# Unified Government Policies Platform --- India

**A single, user-friendly portal to discover, compare and apply to
Indian government schemes.**

------------------------------------------------------------------------

## About

The **Unified Government Policies Platform --- India** is a centralized
web portal that aggregates government schemes, makes them searchable and
comparable, and provides clear, step-by-step details for citizens to
apply.

------------------------------------------------------------------------

## Features

-   Unified list of government schemes with concise overviews.
-   Advanced filters (sector, ministry, eligibility, date, region).
-   Detailed scheme pages: overview, benefits, eligibility, required
    documents, application steps, official link.
-   Instant eligibility checks and document lists.
-   Secure user authentication (JWT + bcrypt).
-   Mobile responsive UI and fast search results.

------------------------------------------------------------------------

## Tech Stack

-   **Frontend:** HTML, CSS, JavaScript, React.js
-   **Backend:** Node.js, Express.js
-   **Database:** MongoDB
-   **Tools:** Git, GitHub, VS Code
